--- title: Sidebar JavaScript slug: sidebar/javascript path: ../ --- {% extends "layouts/default.html" %} {% block page %} {% markdown %} # JavaScript > This document presents the complete JavaScript reference for programmatic access to the Sidebar JavaScript library included with Bootstrap Layout. For basic usage, see the [installation guide](../index.html) and [getting started](index.html). ## Methods ### Sidebar.init > Initializes all the sidebars on page. {% endmarkdown %}
{% markdown %} ```js BootstrapLayout.Sidebar.init() ``` This method has no arguments and it should be called after the page is ready. {% endmarkdown %}
{% markdown %} ### Sidebar.show > Displays a hidden sidebar. {% endmarkdown %}
{% markdown %} ```js BootstrapLayout.Sidebar.show(sidebar, transition) ``` {% endmarkdown %}
Argument Type Required Default value
sidebar

String|jQuery
The argument can be a String representing the sidebar id attribute, for example #my-sidebar or a jQuery element i.e. $('#my-sidebar')

check close
transition

Boolean
Display the sidebar using a simple CSS transition

close true
