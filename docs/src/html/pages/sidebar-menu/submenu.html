Basic usage

To create a basic sidebar submenu:
  • add an ul.sidebar-submenu within a top-level li.sidebar-menu-item
  • submenu items use the .sidebar-menu-item class
  • submenu links use the .sidebar-menu-button class
  • optional add the .open class to any top-level .sidebar-menu-item to display the submenu

Behaviour

See sidebar collapse and sidebar dropdown.
CSS Class Description
.sidebar-menu The sidebar menu wrapper (unordered list)
.sidebar-submenu The sidebar submenu wrapper (unordered list placed within a top level .sidebar-menu-item)
.sidebar-menu-item The sidebar menu item and sidebar submenu item (unordered list item)
.sidebar-menu-button The sidebar menu link and sidebar submenu link
